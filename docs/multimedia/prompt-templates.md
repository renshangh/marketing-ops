# Prompt Templates

## Purpose
Keep reusable prompt skeletons for image and video production.

## Image prompt skeleton
Create a clean, modern social graphic background for [GRAPHIC TYPE].
Use a minimal composition, one accent color, and strong contrast.
Leave clear empty space for overlay text.
Keep the scene technical, brand-safe, and mobile readable.

## Image refinement prompt
Keep the layout the same, but simplify the background, increase contrast, and leave more clean space for text.

## Video concept skeleton
Create a short-form video concept for [TOPIC].
Start with the conflict immediately.
Keep the first 5 to 15 seconds fast and clear.
Define the caption style, beat structure, and call to action.

## Video review prompt
Review this short-form video for hook strength, pacing, caption clarity, and mobile readability.
Focus on the opening 5 to 15 seconds and note any silence, drift, or weak framing.
