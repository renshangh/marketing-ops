# Video Workflow Template

## Purpose
Create a repeatable short-form video workflow for OpenClaw social content.

## Goal
Turn a post idea into a video plan that is ready for scripting, editing, and review.

## Inputs
- topic or angle
- audience
- desired platform
- core hook
- proof point or story point
- caption style
- source footage or concept notes

## Workflow
1. define the hook
2. identify the conflict or tension
3. choose the format
4. map the first 5 to 15 seconds
5. draft the beat structure
6. define on-screen captions
7. review the frame pacing
8. produce or edit the clip
9. extract review frames if needed
10. finalize the export

## Hook rules
- start with the problem, not the intro
- use a conflict-first opening
- remove dead air
- keep the first line immediate and specific

## Beat map
Use a simple beat structure:
- beat 1, hook
- beat 2, context
- beat 3, core point
- beat 4, example or proof
- beat 5, close or CTA

## Caption rules
- keep captions large
- keep them high-contrast
- keep them centered and mobile readable
- avoid dense paragraphs

## Review rules
- inspect the first 5 to 15 seconds first
- check for pacing drift
- check for readability
- check for unnecessary silence
- check for weak visual framing

## Output
The final output should include:
- video concept
- hook line
- beat map
- caption guidance
- review notes
- export or post notes

## Notes
If the video needs deeper production, split the work into:
- concept draft
- script draft
- frame review
- edit revision
