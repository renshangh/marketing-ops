# Localized Video Workflow

## Purpose
Document how to create localized versions of a motion clip.

## Standard flow
1. translate the visual copy if needed
2. translate the narration script
3. choose a matching localized voice
4. generate the MP3 narration
5. render the motion base from the localized graphic
6. merge audio and video with ffmpeg
7. review sync, readability, and timing

## Example
Use the Chinese graphic and Chinese narration for a localized version of the inbound lead capture clip.
For the male-voice variant, use Sinji for the Chinese narration.

## Notes
- keep the same motion style as the source clip
- keep the duration inside the standard clip range
- make sure the audio and the visual language match the audience
